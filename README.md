# Cicekyolla OS

Türkiye'nin yeni nesil çiçek, teslimat, CRM ve SEO işletim sistemi.

## Modüller

- CRM
- Sipariş Merkezi
- Müşteri Yönetimi
- Kurye Yönetimi
- Çelenk Yönetimi
- SEO Merkezi
- Yapay Zeka Merkezi
- Finans Merkezi
- Bayilik Sistemi

## Teknoloji

- Next.js
- PostgreSQL
- Redis
- Meilisearch
- Cloudflare
- Docker

## Durum

Planlama ve mimari aşaması.
