# CICEKYOLLA OS

## Project Vision

Türkiye'nin en gelişmiş çiçek, teslimat, CRM ve SEO işletim sistemi oluşturulacaktır.

Bu sistem:

* Çiçek E-Ticaret Platformu
* CRM Sistemi
* Sipariş Yönetim Sistemi
* Kurye Yönetim Sistemi
* Çelenk Operasyon Sistemi
* SEO Yönetim Sistemi
* Yapay Zeka Merkezi
* Bayilik Sistemi
* Kurumsal Satış Sistemi

modüllerini tek merkezde birleştirecektir.
